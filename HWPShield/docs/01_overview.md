# 01. 프로젝트 개요 (Project Overview)

## 프로젝트 이름

**HWPShield**  
*HWP + Shield (한글 문서 방패)*

---

## 프로젝트 목표

HWPShield는 웹 기반 한글 문서(.hwp/.hwpx) 악성코드 분석 도구입니다. 사용자가 문서를 업로드하면 상세한 악성코드 분석 리포트를 제공하며, 실제 APT(Advanced Persistent Threat) 캠페인에서 사용되는 공격 벡터를 탐지합니다.

### 주요 대상 위협

이 프로젝트는 다음 APT 그룹들이 한국 정부 및 기관을 대상으로 사용한 공격 패턴을 탐지합니다:

| APT 그룹 | 활동 기간 | 특징 |
|----------|----------|------|
| **APT37 (ScarCruft)** | 2012-현재 | 북한 계열, EPS 취약점 선호 |
| **APT38** | 2014-현재 | 금융 기관 대상, SWIFT 공격 |
| **Kimsuky** | 2013-현재 | 한국 정부 기관 대상, 스피어피싱 |
| **RedEyes** | 2021-현재 | M2RAT, 스테가노그래피 사용 |

---

## 핵심 기능

### 1. 정적 분석 (Static Analysis)

- 문서를 **절대 실행하지 않고** 내부 구조만 분석
- OLE2 파일 형식 파싱
- 압축된 스트림(zlib) 해제 및 검사

### 2. 다중 탐지 모듈

6개의 독립적인 탐지 모듈이 각각 다른 공격 패턴을 탐지:

1. **EPS/PostScript 탐지** - CVE-2017-8291 (GhostButt)
2. **OLE 객체 탐지** - 임베디드 실행 파일
3. **스크립트 분석** - 매크로 및 VBE/VBS
4. **IOC 추출** - URL, IP, 파일 경로
5. **스테가노그래피 탐지** - 이미지에 숨겨진 페이로드
6. **구조적 이상 탐지** - 메타데이터 및 파일 구조 분석

### 3. 리스크 스코어링

누적 점수 기반의 체계적인 위험도 평가:

| 위험 등급 | 점수 범위 | 색상 |
|----------|----------|------|
| **CLEAN** | 0-14 | 녹색 |
| **SUSPICIOUS** | 15-34 | 노란색 |
| **HIGH RISK** | 35-59 | 주황색 |
| **MALICIOUS** | 60+ | 빨간색 |

### 4. 사용자 친화적 리포트

- 한국어/영어 지원
- 다크,화이트 모드 지원
- JSON 형식 다운로드
- IOC 목록 복사 기능

## 기술 스택

### 프론트엔드

| 기술 | 버전 | 용도 |
|------|------|------|
| React | 18.x | UI 프레임워크 |
| TypeScript | 5.x | 타입 안전성 |
| Tailwind CSS | 3.x | 스타일링 |
| Axios | - | HTTP 클라이언트 |

### 백엔드

| 기술 | 버전 | 용도 |
|------|------|------|
| Python | 3.11+ | 런타임 |
| FastAPI | 0.100+ | 웹 프레임워크 |
| olefile | 0.47+ | OLE 파싱 |
| Uvicorn | - | ASGI 서버 |

### 파일 파싱

- **olefile**: OLE2 파일 구조 분석
- **zlib**: 스트림 압축 해제 (wbits=-15)
- **struct**: 바이너리 데이터 파싱
- **hashlib**: MD5/SHA256 해시 계산

### 배포

- Docker 컨테이너화
- docker-compose 통합 배포

## 프로젝트 범위

### 포함 기능

- [1] .hwp 파일 분석
- [2] .hwpx 파일 분석 (추후 지원)

- [4] 리스크 스코어링 시스템

- [7] 다국어 지원
- [8] Docker 배포

- [1] 실시간 모니터링
- [2] 샌드박스 동적 분석
- [3] YARA 룰 통합
- [4] 클라우드 스케일링

## 성공 기준

1. **정확성**: 알려진 악성 HWP 샘플 100% 탐지
2. **성능**: 10MB 파일 분석 5초 이내
3. **안전성**: 분석 중 파일 실행 절대 불가
4. **사용성**: 직관적인 UI, 명확한 리포트
